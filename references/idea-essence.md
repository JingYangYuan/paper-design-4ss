# 研究问题生成方法论

> 与 `frame/` 七大学科框架库配套的 RQ 生成、评估与精炼体系。

---

## 一、从理论到 RQ：五条生成路径

### 路径 A：理论驱动（从框架到问题）

1. 从 `frame/` 中选择 1-3 个候选理论
2. 提取该理论的**核心机制链**（X → M → Y）
3. 识别机制链中**未被充分检验**的环节
4. 将待检验环节转化为可操作的研究问题

**模板**：`在[人群]中，[理论 T 的机制 M]是否解释了[X]与[Y]之间的关系？`

### 路径 B：空白驱动（从文献缺口到问题）

1. 定位 frame 文件中标注的**理论竞争/未解决争论**
2. 锁定"预测相矛盾但未被裁定"的理论对
3. 提出可区分两种预测的经验检验

**模板**：`[理论A]预测[X→Y]为正，而[理论B]预测为负；在[条件C]下，哪种预测成立？`

### 路径 C：嫁接驱动（从跨学科连接到问题）

1. 识别学科 A 中理论 T1 与学科 B 中理论 T2 的**概念同构**或**机制互补**
2. 验证嫁接的适配度（前提条件是否在目标领域成立）
3. 推导可检验的跨学科命题

**模板**：`[学科A的概念/机制]移植到[学科B的现象]后，能否导出现有理论未能预测的新假设？`

### 路径 D：边界驱动（从调节条件到问题）

1. 选定一个已确立的 X→Y 关系
2. 从 frame 文件的理论边界条件中提取候选调节变量 W
3. 提出"关系在 W 的不同水平上是否异质"的问题

**模板**：`[X→Y]的关系是否因[W]而异？[理论T]预测的边界条件是否成立？`

### 路径 E：数据驱动（从可用变量到问题）

1. 盘点目标数据集中的关键变量（Y, X, M, W, C, Z）
2. 将有理论基础的变量映射到 frame 中的机制链
3. 优先匹配"数据中有独特测量"+"理论上有空白"的变量对

## 二、RQ 公式库（按研究范式）

| 范式 | 公式模板 |
|------|---------|
| **因果-机制** | 在[人群]中，[X]是否通过[M]影响[Y]？[理论T]指定的机制能否被经验检验？ |
| **调节/边界** | [X→Y]的关系在[W]的不同水平上是否异质？[理论T]的边界条件是否成立？ |
| **竞争理论** | [理论A]预测[X→Y]而[理论B]预测相反方向；在[研究设计]下，哪种预测得到支持？ |
| **跨层次** | [宏观层Z]如何调节[微观层X→Y]的关系？跨层次机制是什么？ |
| **时间变化** | [X→Y]的关系从[t1]到[t2]如何变化？[理论T]能否解释这一时间动态？ |
| **比较** | [X→Y]的关系在[群体A]和[群体B]之间有何不同？[理论T]能否解释差异？ |
| **政策/干预** | [政策/干预P]对[Y]产生了什么影响？[理论T]指定的机制是否中介了这一效应？ |

## 三、七学科高频 RQ 起手式

| 学科 | 典型 RQ 起手式 |
|------|---------------|
| **社会学** | `[分层过程X]如何通过[机制M]影响[不平等结果Y]？` |
| **公共管理** | `[制度安排X]如何影响[公共服务结果Y]？[组织机制M]是否中介？` |
| **心理学** | `[个体差异/情境X]如何通过[认知/情感机制M]影响[行为Y]？` |
| **传播学** | `[媒介暴露X]如何通过[认知/情感路径M]影响[态度/行为Y]？` |
| **经济学** | `[政策/冲击X]对[经济结果Y]的因果效应是什么？[机制M]如何解释？` |
| **教育学** | `[教学/制度条件X]如何影响[学习结果Y]？[认知/动机机制M]是什么？` |
| **政治学** | `[制度/权力结构X]如何影响[政治结果Y]？[行为/规范机制M]是否中介？` |

## 四、RQ 质量四维评估

| 维度 | 1-3 (弱) | 4-6 (可) | 7-8 (强) | 9-10 (突出) |
|------|----------|----------|----------|------------|
| **理论锚定** | 无明确理论来源 | 提及理论但连接模糊 | 明确指定理论+机制链 | 裁定竞争理论或填补命名空白 |
| **可检验性** | 概念不可操作化 | 可操作化但识别策略弱 | 有明确数据和识别策略 | 自然实验/IV/RD 可用 |
| **新颖性** | 已饱和 | 增量扩展 | 新情境/新机制 | 未探索领域 |
| **实质意义** | 纯学术兴趣 | 间接启示 | 直接回应学科争论 | 政策/实践变革潜力 |

**门控规则**：理论锚定 < 4 或可检验性 < 4 → 返回重述 RQ。

## 五、常见失败模式

| 失败模式 | 表现 | 修复 |
|---------|------|------|
| **无理论锚定** | RQ 仅描述"X 与 Y 的关系"而不指定理论机制 | 从 frame 中至少匹配一个理论，明确其机制链 |
| **概念悬浮** | 核心概念在 frame 理论中找不到对应 | 检查概念是否可追溯到 frame 中的理论构念 |
| **框架拼凑** | 机械堆砌多个理论而无论证其兼容性 | 使用 storm-patterns.md 的嫁接策略验证兼容性 |
| **数据盲目** | 提出 RQ 后才发现关键变量无数据来源 | 在最终确定 RQ 前验证数据可行性 |
| **学科错配** | RQ 的理论来源与宣称的学科框架不一致 | 核对 SKILL.md 的路由表，确保学科归属正确 |
