# scholar-idea 方法论精髓

## 一、RQ 公式库

将模糊想法转化为正式研究问题的标准化句式:

| 形式 | 公式 | 适用场景 |
|------|------|---------|
| **因果-机制** | 在 [人群] 中，[X] 如何通过 [M] 影响 [Y]？ | 理论指定了完整的 X→M→Y 链条 |
| **比较** | [X]→[Y] 的关系在 [群体A] 和 [群体B] 之间有何不同？为什么？ | 跨群体/跨情境比较 |
| **政策评估** | [政策/干预X] 对 [目标人群] 的 [Y] 产生了什么影响？相对于 [反事实]？ | 政策/干预效果评估 |
| **组织** | [组织特征X] 如何在 [条件] 下影响 [成员/客户] 的 [Y]？ | 组织社会学 |
| **空间** | [地域条件X] 如何影响 [Y]？效应在哪些人群中最强？ | 城市/社区/环境社会学 |
| **时间变化** | [X]→[Y] 的关系从 [t1] 到 [t2] 如何变化？什么因素解释了这一变化？ | 面板/重复截面数据 |
| **调节** | [X]→[Y] 的关系在什么条件下增强/减弱？[W] 如何调节这一效应？ | 理论指定了边界条件 |
| **涌现** | [宏观模式Y] 如何从 [微观行为X] 中涌现？涌现的条件是什么？ | 计算/ABM/多层次 |

## 二、学科领域模式速查

### 2.1 不平等与社会流动
- **典型 Y**: 收入/财富/教育获得/健康/监禁风险
- **典型 X**: 阶层出身/邻里剥夺/劳动力市场不稳定/学校质量
- **核心机制**: 资源可及性、制度把关、网络闭合、压力负担
- **RQ 起手**: `[分层过程X] 如何通过 [机制M] 影响 [流动结果Y]？家庭背景的作用被控制后效应是否仍然存在？`

### 2.2 移民与融入
- **典型 Y**: 劳动力市场结果/语言习得/公民参与/归属感
- **典型 X**: 法律身份/目的地情境/歧视/同族网络
- **核心机制**: 选择性暴露、信息摩擦、学历认证、社会融入
- **RQ 起手**: `[制度/法律情境X] 如何通过 [机制M] 影响 [移民/代际] 的 [融入结果Y]？`

### 2.3 健康与社会决定因素
- **典型 Y**: 发病率/心理健康/预防性医疗使用/死亡风险
- **典型 X**: 隔离/污染/医疗可及性/就业不稳定
- **核心机制**: 慢性压力、物质剥夺、医疗可及路径、行为适应
- **RQ 起手**: `[社会/空间暴露X] 如何通过 [路径M] 影响 [健康结果Y]？谁受到的影响最大？`

### 2.4 政治行为与公共舆论
- **典型 Y**: 投票/抗议/信任/极化/政策态度
- **典型 X**: 媒体暴露/网络构成/压制/地方冲击
- **核心机制**: 信息扩散、身份激活、不满框架、效能感
- **RQ 起手**: `[信息/社会暴露X] 如何影响 [政治结果Y]？效应在什么条件下最强？`

### 2.5 组织、工作与技术
- **典型 Y**: 招聘结果/晋升/生产力/倦怠/离职
- **典型 X**: 算法管理/远程政策/监控强度/团队结构
- **核心机制**: 自主性、评估透明度、协调成本、偏见放大
- **RQ 起手**: `[组织/技术实践X] 如何影响 [工作者结果Y]？效应因哪些工作者特征而异？`

### 2.6 家庭、性别与生命历程
- **典型 Y**: 生育/婚姻转变/照护负担/收入惩罚
- **典型 X**: 政策情境/家庭内协商/劳动灵活性/性别规范
- **核心机制**: 机会成本、社会规范、时间稀缺、制度支持
- **RQ 起手**: `[制度/家庭情境X] 如何影响 [生命历程结果Y]？效应如何因性别/阶层/种族而异？`

### 2.7 教育与分层
- **典型 Y**: 学业成就/大学入学/专业选择/坚持率/债务负担
- **典型 X**: 分流/学校资源/教师期望/同伴构成
- **核心机制**: 累积优势、标签效应、社会资本、制度分类
- **RQ 起手**: `[学校组织条件X] 如何影响 [教育结果Y]？效应在哪些学生中最强？`

### 2.8 犯罪、警务与法律
- **典型 Y**: 逮捕风险/量刑严厉度/累犯/法律信任
- **典型 X**: 执法强度/邻里监视/政策改革
- **核心机制**: 威慑、选择性执法、法律犬儒主义、程序正义
- **RQ 起手**: `[司法实践/政策X] 对 [法律结果Y] 有什么影响？效应是否因群体而异？`

### 2.9 种族、族群与不平等
- **典型 Y**: 居住隔离/种族财富差距/健康差异/招聘歧视/教育差距
- **典型 X**: 种族分类/肤色/邻里构成/歧视暴露/移民执法
- **核心机制**: 结构性种族主义、累积劣势、居住分类、统计歧视、社会闭合
- **RQ 起手**: `[种族/族群分层过程X] 如何通过 [制度机制M] 影响 [结果Y]？`
- **异质性提示**: 按移民身份、代际、肤色、性别、阶层分解

### 2.10 社会运动与集体行动
- **典型 Y**: 抗议参与/运动兴衰/政策变化/公众舆论变化
- **典型 X**: 政治机会结构/组织资源/不满框架/媒体报道/压制
- **核心机制**: 资源动员、政治中介、框架对齐、网络扩散、旁观者激活
- **RQ 起手**: `在什么条件下 [社会运动X] 导致了 [政治/政策结果Y]？什么组织/情境因素中介了这一关系？`

### 2.11 文化、认同与语言
- **典型 Y**: 语言变异/身份表达/地位评价
- **典型 X**: 互动场景/受众构成/平台规范
- **核心机制**: 指示性、受众设计、声望/污名、边界工作
- **RQ 起手**: `[社会情境X] 如何调节 [语言/文化实践Y]？什么身份在这个过程中被协商？`

### 2.12 环境与气候
- **典型 Y**: 环境暴露/气候风险感知/绿色行为/环境健康差异
- **典型 X**: 工业选址/种族-阶层分布/碳定价/气候传播/灾害经历
- **核心机制**: 生产跑步机、环境正义 (不成比例负担)、风险文化认知、社区能力
- **RQ 起手**: `[社会/结构条件X] 如何影响 [环境暴露/气候行为Y]？谁承担了不成比例的风险？`

## 三、机制菜单

每 RQ 选 1 个主要机制 + 1 个次要机制:

| 机制通道 | 核心逻辑 | 典型变量 |
|---------|---------|---------|
| **资源通道** | 金钱、时间、信息、制度支持的获取差异 | 收入、财富、时间使用、信息素养 |
| **网络通道** | 暴露、扩散、中介、闭合、同质性 | 社交网络指标、弱/强关系数量 |
| **制度通道** | 规则、把关、制裁、行政负担 | 政策资格、官僚程序、准入门槛 |
| **文化通道** | 意义、规范、污名、合法性 | 文化资本、污名感知、价值取向 |
| **心理通道** | 压力、效能感、归属感、威胁感知 | 自尊、控制感、歧视感知 |
| **空间通道** | 距离衰减、隔离、环境约束 | 邻里特征、通勤距离、污染暴露 |
| **时间通道** | 序列、路径依赖、累积劣势 | 事件史、队列标识、暴露时长 |

**机制陈述模板**:
`我们预期 [X] 主要通过 [机制] 影响 [Y]，因为 [简短因果逻辑]。`

## 四、数据与设计匹配

### 4.1 快速匹配表

| RQ 类型 | 推荐设计 | 推荐数据 |
|---------|---------|---------|
| 强因果政策问题 | 行政面板 + DiD/事件研究 | 行政记录、政策面板 |
| 处理阈值问题 | RD (精确断点) | 有明确断点的行政数据 |
| 个体变化问题 | 面板 FE | 追踪调查 (PSID, CFPS, CHARLS) |
| 扩散/网络问题 | 网络模型 | 关系数据 |
| 文本/意义问题 | NLP/质性编码 | 语料库/访谈转录 |
| 机制丰富但 N 小 | 混合方法 (量化+访谈) | 调查+嵌套访谈 |

### 4.2 可行性警示

- 关键 X 测量弱 → 降级为探索性/关联性框架
- 混淆变量多不可观测 → 提出 IV、自然实验或敏感性界值
- 时间线短 → 优先选择已知变量质量的二手数据

## 五、新颖性检查

### 5.1 四维评估

| 维度 | 关键问题 |
|------|---------|
| **理论增量** | 是否裁定竞争解释？是否连接了通常不对话的文献？ |
| **经验增量** | 新数据？新情境？新时间段？比先前研究更精确的测量？ |
| **识别增量** | 比先前研究更好的因果杠杆（自然实验/新IV/面板）？ |
| **实质重要性** | 答案对核心社会科学争论或政策有影响吗？ |

### 5.2 新颖性威胁分类

- **已饱和** (SATURATED): 3+ 高质量研究回答相同问题 → 需要转向或大幅差异化
- **增量** (INCREMENTAL): 先前工作存在，但在人群/时间/方法上有空白 → 明确扩展理由
- **空白** (GAP): 问题被提出但未被良好回答 → 强切入点
- **未探索** (UNEXPLORED): 问题未被直接研究 → 高风险高回报，检查空白是否反映困难而非忽视

**若四维全为 中/低 → 在继续之前修订 RQ。**

## 六、文献扫描查询模板

| 查询类型 | 模板 |
|---------|------|
| **主题+结果** | `"[主题]" "[结果变量]" sociology OR demography` |
| **机制** | `"[提出的机制]" "[结果]" "[人群]"` |
| **识别/方法** | `"[主题]" "panel data" OR "natural experiment" OR "difference-in-differences"` |
| **时效性** | `"[主题]" "[结果]" after:2021` |
| **综述** | `"[主题]" "systematic review" OR "meta-analysis"` |
| **跨国比较** | `"[主题]" "cross-national" OR "comparative" OR "OECD"` |
| **最接近竞争者** | `"[X] effect on [Y]" "[目标人群]"` |

**搜索优先级**: 本地文献库 → Google Scholar → Semantic Scholar → CNKI (中文文献) → WebSearch (补缺)

## 七、常见失败模式与修复

| 失败模式 | 表现 | 修复 |
|---------|------|------|
| **Y 过于宽泛** | "不平等" 但无具体可测量结果 | 缩小 Y: 收入不平等/健康差距/教育差距 |
| **无机制** | 只问 "是否" 不问 "如何/为什么" | 添加至少一个机制通道 |
| **数据不匹配** | 用弱的截面数据问因果问题 | 匹配设计强度与声明强度 |
| **范围模糊** | 无人群/情境/时间边界 | 声明范围: "在 [人群] 中 [时间] 的 [情境]" |
| **平凡扩展** | 多一个控制变量但无新理论杠杆 | 确认理论增量后再推进 |
| **假定新颖性** | 未检索就声称 "未被研究过" | 先扫描文献再声明新颖性 |
| **数据盲目** | 不知道关键变量在任何数据集中是否存在 | 确认关键变量存在于一个命名的、可及的数据集中 |

## 八、主要社会科学数据集速查

### 国际调查

| 数据集 | 覆盖 | 分析单位 | 关键变量 |
|--------|------|---------|---------|
| **WVS** (世界价值观调查) | 1981–今, ~100国 | 个体 | 价值观、信念、政治态度 |
| **ISSP** (国际社会调查) | 1985–今, 40+国 | 个体 | 工作、家庭、宗教、不平等 |
| **LIS** (卢森堡收入研究) | 1970s–今, 50+国 | 家庭 | 收入、财富、社会转移——跨国标准化 |
| **PISA** (国际学生评估) | 2000–今, OECD | 学生 | 学业成就、学校氛围、家庭背景 |

### 中国调查

| 数据集 | 覆盖 | 分析单位 | 关键变量 |
|--------|------|---------|---------|
| **CFPS** (中国家庭追踪调查) | 2010–今, 全国 | 个体/家庭/社区 | 收入、教育、健康、迁移、家庭结构 |
| **CGSS** (中国综合社会调查) | 2003–今, 全国 | 个体 | 社会态度、阶层、不平等、政治参与 |
| **CHARLS** (中国健康与养老追踪) | 2011–今, 45+ | 个体/家庭 | 健康、退休、养老金、代际转移 |
| **CLDS** (中国劳动力动态调查) | 2012–今, 全国 | 个体/家庭/社区 | 就业、收入、劳动条件、迁移 |
| **CHFS** (中国家庭金融调查) | 2011–今, 全国 | 家庭 | 资产、负债、收入、支出、金融行为 |
| **CSS** (中国社会状况综合调查) | 2006–今, 全国 | 个体 | 社会态度、生活质量、社会问题 |

### 美国主要调查

| 数据集 | 覆盖 | 分析单位 | 关键变量 |
|--------|------|---------|---------|
| **GSS** (综合社会调查) | 1972–今 | 个体 | 态度、宗教、政治、阶层、种族 |
| **ACS** (美国社区调查) | 2005–今 | 个体/家庭 | 收入、教育、就业、住房、移民、语言 |
| **PSID** (收入动态追踪) | 1968–今 | 家庭/个体 | 收入、财富、就业、家庭、健康——50+年追踪 |
| **Add Health** (青少年健康追踪) | 1994–今 | 个体 | 健康、同伴网络、教育、家庭——青少年→成年 |
| **NHIS** (全国健康访谈) | 1957–今 | 个体 | 自评健康、慢性病、活动限制、医疗可及性 |
