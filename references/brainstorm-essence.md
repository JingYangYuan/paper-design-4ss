# scholar-brainstorm 方法论精髓

---

## 一、输入类型检测

| 输入类型 | 检测信号 | 提取重点 |
|---------|---------|---------|
| **编码簿** | 变量名、值标签、跳转模式 | 变量清单、测量尺度、跳转逻辑(揭示子人群) |
| **问卷** | 题目主干、选项、章节标题 | 测量构念、题目顺序(揭示理论模型)、筛选条件 |
| **数据文件** (.csv/.dta/.rds) | 列名、数据类型、分布 | 变量名、缺失模式、样本量、分析单位、面板结构 |
| **数据字典** | 变量名+标签+类型+取值 表格 | 同编码簿但更简洁 |
| **已发表论文** (.pdf/DOI) | 标题/摘要/引言/方法 | 核心概念、主要发现、未解决问题、局限性 |

## 二、变量角色分类体系

| 角色 | 英文 | 定义 | RQ 位置 | 识别规则 |
|------|------|------|---------|---------|
| **结果变量** | Y (Outcome) | 需解释的状态、行为或事件 | 因变量 | 在焦点时间点测量；通常是"状态"类变量 |
| **预测变量** | X (Predictor) | 理论指定的原因/暴露/条件 | 自变量 | 在焦点时间点之前测量 |
| **机制变量** | M (Mediator) | X→Y 的中介过程 | 中介变量 | 在 X 之后、Y 之前测量 |
| **调节变量** | W (Moderator) | 改变 X→Y 关系强度的条件 | 交互项 | 通常是人口学特征或情境变量 |
| **混淆变量** | C (Confounder) | X 和 Y 的共同原因 | 控制变量 | 同时影响 X 和 Y；在 X 之前确定 |
| **情境变量** | Z (Context) | 更高层次的地理/制度条件 | 层次-2 变量 | 地理/制度标识符 |
| **人口变量** | D (Demographic) | 背景特征 | 分层/分组 | 年龄/性别/种族等 |

**赋值启发规则**:
- 焦点时间点**之前**测量的变量 → 可能是 X / C / D
- 焦点时间点**当时**测量的变量 → 可能是 Y / M / W
- "状态"类变量 (收入、健康) → 偏向 Y
- "条件/暴露"类变量 → 偏向 X
- "态度/感知"类变量 → 可能是 M 或 Y (取决于 RQ)
- 地理/制度标识符 → 偏向 Z

## 三、RQ 生成六策略

### 策略 A: Y 驱动 (结果变量先行)
1. 识别最有趣/最具政策相关性的 Y
2. 对每个 Y，扫描有理论基础的 X
3. 寻找连接 X→Y 的机制变量 M
4. 检查可能产生异质性的调节变量 W

### 策略 B: X 驱动 (预测变量先行)
1. 识别数据集中独特或未充分利用的 X ("星标变量")
2. 对每个 X，头脑风暴其可能影响的 Y
3. 绘制机制链条和调节条件

### 策略 C: 空白驱动 (填补缺失环节)
1. 识别理论上相关但从未同时研究的变量对
2. 寻找可填补理论空白的机制变量
3. 优先关注能提供因果识别的变量 (面板结构、自然实验、工具变量)

### 策略 D: 异质性驱动 (子群变异)
1. 识别人口/情境调节变量 W, Z
2. 对已建立的 X→Y 关系，问："这种关系因 W 而异吗？"
3. 优先关注交叉性组合 (种族×性别、阶层×移民身份)

### 策略 E: 时间/变化驱动
1. 面板或重复截面：识别多次测量的变量
2. 问："X→Y 关系如何随时间变化？"
3. 寻找队列效应、时期效应或政策断点

### 策略 F: 方法创新驱动
1. 识别使更强因果识别成为可能的变量
2. 寻找自然实验、工具变量、数据中的断点
3. 问："我能否用更好的因果杠杆回答一个老问题？"

**星标变量 (Star Variables)** — 值得特别关注的变量:
- **独特性**: 仅在少数数据集中可用 (biomarkers、社交网络数据)
- **测量质量**: 测量方式优于其他数据集 (多项目量表 vs 单题)
- **因果杠杆**: 提供更强的因果识别机会 (断点、IV、自然实验)
- **政策相关性**: 具有直接的政策或实践含义

## 四、RQ 公式库

| 形式 | 模板 |
|------|------|
| **直接效应** | 在 [数据集] 中，[X] 与 [Y] 之间存在怎样的关联？[可用混淆变量C] 控制后该关联如何变化？ |
| **机制** | [M] 是否中介了 [X]→[Y] 的关系？总效应中有多大比例通过这一路径？ |
| **异质性** | [X]→[Y] 的关系是否因 [W: 种族/性别/阶层/队列/地域] 而异？差异的来源是什么？ |
| **时间变化** | [X]→[Y] 的关系在 [时期/波次] 之间如何变化？什么结构因素导致了这一变化？ |
| **比较** | [X]→[Y] 的关系在 [群体A] 和 [群体B] 之间有何不同？[M变量] 是否能解释这一差异？ |
| **分解** | [群体A] 与 [群体B] 在 [Y] 上的差距中，多大比例可归因于 [X特征] 的差异 vs 特征回报率的差异？ |
| **跨层次** | [Z: 地域/制度层变量] 如何调节个体层面的 [X]→[Y] 关系？ |

## 五、七种谜题模板

| 谜题类型 | 模板 | 示例 |
|---------|------|------|
| **异常** | [X] 根据 [理论] 应预测 [Y]，但数据显示 [意外模式] | 教育增加收入但不增加群体 Z 的生活满意度 |
| **趋势背离** | [Y1] 和 [Y2] 应同步变化，但在 [事件/时期] 后出现背离 | 结婚率下降但同居率未完全补偿 |
| **意外零效应** | 尽管有强理论预期，[X] 对 [子群] 的 [Y] 无影响 | 社区效应在第二代移民中消失 |
| **反转** | [X]→[Y] 的关系在 [子群 W] 中出现符号反转 | 教育回报对无证工人为负 |
| **机制错配** | 预期机制 [M1] 不解释 [X]→[Y]；反而是 [M2] 发挥作用 | 歧视→健康通过警惕性而非物质匮乏起作用 |
| **持续存在** | [Y] 的差异在控制了所有可用的 [X, C] 后仍然存在 | 种族财富差距在控制收入、教育、遗产后仍存续 |
| **涌现** | [关系] 仅在通常被聚合的 [子群/情境] 中出现 | 性别薪酬差距仅在特定职业领域出现 |

## 六、变量配对启发

### 高潜力配对
- **新 X × 成熟 Y**: 新测量的暴露 → 充分研究的结果
- **成熟 X × 新 Y**: 充分研究的预测变量 → 未充分研究的结果
- **机制检验**: X 和 Y 已充分确立，数据集独特地拥有 M
- **交叉性调节**: 数据集同时有 种族 + 阶层 + 性别 → 交互分析
- **跨层次**: 数据将个体变量与地域情境 (区县/省市) 关联
- **纵向杠杆**: 同一变量在 T1 和 T2 均有测量 → 个体内/固定效应

### 低潜力配对 (避免)
- 两个人口变量 (如年龄和种族) 且无明确因果方向
- 同义反复关系 (如"就业"预测"收入")
- 目标子人群中缺失率 >70% 的变量
- 需要面板数据但仅有截面的因果关系
- 高度相关 (r > 0.8) 的变量对

## 七、常见陷阱

| 陷阱 | 表现 | 纠正 |
|------|------|------|
| **数据捕捞** | 无理论动机地检验所有可能的 X→Y 对 | 每个 RQ 都扎根于特定的理论谜题或争论 |
| **HARKing** | 先探索数据发现结果，再编造理论 | 在分析前预承诺 RQ；诚实地标注探索性发现 |
| **测量混淆** | 将代理变量等同于构念本身 | 明确指出变量实际测量了什么 vs 你声称它捕捉了什么 |
| **因果过度推断** | 从截面观测数据得出因果论断 | 语言匹配设计: 截面用"相关"，准实验才用"效应" |
| **厨房水槽模型** | 无理论依据地纳入所有可用控制变量 | 使用 DAG 决定该控制什么、不该控制什么 (碰撞偏误) |
| **子群选择性报告** | 检验20个子群，只报告显著的2个 | 预指定子群；多重比较校正；报告所有检验 |
| **忽略选择机制** | 未考虑谁在样本中以及为什么 | 绘制选择过程: 谁回应？谁流失？总体是谁？ |

## 八、评估与排序协议

### DATA 模式 (6 维评分)
| 维度 | 权重 | 说明 |
|------|------|------|
| 新颖性 | 20% | 已饱和 / 增量 / 空白 / 未探索 |
| 数据就绪度 | 15% | 关键变量是否全部可用且测量良好 |
| 理论显著性 | 20% | 是否回应活跃争论、填补命名空白、检验机制 |
| 识别强度 | 15% | 数据是否支持可信的因果论断 |
| 发表潜力 | 10% | 与目标期刊范围和规范的匹配度 |
| **经验信号** | 20% | 双变量检验中的效应量和显著性 |

### MATERIAL/PAPER 模式 (5 维评分)
同上但去掉经验信号维度，理论显著性升至 25%，新颖性升至 25%。

### 新颖性威胁评级
- **已饱和**: ≥3篇以相同方法回答了相同RQ → 放弃或大幅差异化
- **增量**: 1-2篇接近但人群/时间/方法不同 → 可用，需明确贡献陈述
- **空白**: 论文涉及部分但非完整因果链 → 强潜力
- **未探索**: <1篇涉及任何组分 → 高新颖性，验证可行性
