# scholar-design 方法论精髓

## 一、设计选择决策树

| 研究目标 | 数据结构 | 处理分配 | 推荐设计 | 期刊匹配 |
|---------|---------|---------|---------|---------|
| 离散 X 的因果效应 | 面板，重复测量 | 自然变异 | 双向 FE (单位+时间) | ASR, AJS, Demography |
| 政策因果效应 | 面板，渐进实施 | 政策推行 | 交错 DiD (Callaway-Sant'Anna) | ASR, Demography |
| 阈值附近因果 | 截面/面板 | 规则性断点 | RD (精确/模糊) | ASR, AJS, Science Advances |
| 工具变量因果 | 截面/面板 | 外生 IV | 2SLS / IV | AJS, Demography |
| 处理因果效应 | RCT (田野/实验室) | 随机 | OLS / Lin 估计量 | Science Advances, NHB |
| 群集因果 | 群集 RCT (学校/诊所) | 群集随机 | GEE / 多水平+ICC调整 | Science Advances, NHB |
| 歧视/差别对待 | 审计/对应研究 | 配对随机 | 对子内分析 / 条件 logit | ASR, AJS, Science Advances |
| 调查实验因果 | 调查实验 | 随机情景/联合 | AMCE / OLS 聚类 | ASR, AJS, NHB |
| 描述 (分布、趋势) | 截面/面板 | N/A | OLS + 分解 | Demography, AJS |
| 机制/过程 | 质性, N<60 | N/A | 深度访谈/民族志 | AJS, ASR (混合) |
| 大规模分类 | 文本/数字数据 | N/A | ML / NLP | NCS, Science Advances |
| 预测/画像 | 调查/行政数据 | N/A | 正则化回归 / RF | NCS, Science Advances |

## 二、模型选择速查

| 结果变量类型 | 分布 | 推荐模型 | 关键输出 |
|-------------|------|---------|---------|
| 连续 (正态) | Gaussian | OLS | β + 稳健 SE |
| 连续 (偏态) | Log-normal | OLS on log(Y) | 反转换解释 |
| 二元 (0/1) | Bernoulli | Logit | **必须报告 AME** (非 odds ratio) |
| 有序分类 (3-7级) | Ordinal | 有序 Logit | **必须报告 AME** |
| 名义分类 | Multinomial | 多项 Logit | 相对风险比 |
| 计数 (无过度离散) | Poisson | Poisson 回归 | 发生率比 |
| 计数 (过度离散) | NB | 负二项回归 | 发生率比 |
| 计数 (过多零) | Zero-inflated | ZINB / Hurdle | 两部分分别解释 |
| 时间-事件 | Survival | Cox PH | 风险比 + PH 假设检验 |
| 面板-连续 | Gaussian | 双向 FE | 聚类 SE |
| 面板-二元 | Bernoulli | 条件 Logit | 组内效应 |
| 多层次-连续 | Gaussian | 混合模型 (lme4) | ICC + 随机效应方差 |

**关键计量规则**:
1. Logit/有序Logit → **永远报告 AME (平均边际效应)**，不报告原始 log-odds 或 odds ratio
2. 面板数据 → 默认固定效应而非随机效应
3. 聚类 SE → 在最高处理层级聚类
4. R 计算 AME: `marginaleffects::avg_slopes(model)`

## 三、识别策略核心

| 策略 | 核心假设 | 关键检验 | 主要威胁 |
|------|---------|---------|---------|
| **OLS+控制** | 基于可观测变量的选择 | Oster δ; E-value | 遗漏变量偏误 |
| **面板 FE** | 严格外生性 | F 检验 (FE vs POLS) | 时变混淆 |
| **交错 DiD** | 平行趋势 + 无预期 | 事件研究图; 前趋势检验 | 异质性处理效应 |
| **RD (断点)** | 断点附近局部随机 | McCrary 密度检验; 前定变量平衡 | 操纵 running variable |
| **IV/2SLS** | 排他性 + 相关性 | 第一阶段 F>10; 过度识别 | 弱工具; 排他性违规 |
| **匹配** | 共同支撑域 + 无不可观测混淆 | 匹配后平衡 (SMD<0.1) | 不可观测混淆 |
| **审计研究** | 配对随机化 | 信号效度检验 | 非盲态审计员 |
| **RCT** | 随机分配+无干扰 | 平衡检验; 损耗分析 | 损耗; 不依从 |

## 四、功效分析要点

### 4.1 快速公式
```r
library(pwr)
# 两样本 t 检验
pwr.t.test(d = 0.3, sig.level = 0.05, power = 0.80, type = "two.sample")

# 给定 N，求最小可检测效应 (MDES)
pwr.t.test(n = 400, sig.level = 0.05, power = 0.80, type = "two.sample")
```

### 4.2 社会科学效应量基准
| 领域 | 典型效应 | 来源 |
|------|---------|------|
| 教育回报 (每学年) | β ≈ 0.06–0.10 | 劳动经济学 |
| 调查态度实验 | d = 0.10–0.30 | 社会心理学 |
| 歧视审计/场景实验 | d = 0.20–0.50 | 社会学 |
| 社区效应 | d = 0.10–0.25 | 城市社会学 |
| 健康行为干预 | d = 0.20–0.40 | 公共卫生 |

### 4.3 群集设计功效
```
群集 RCT 设计参数:
  DEFF = 1 + (m̄ − 1) × ρ     (设计效应)
  m̄ = 每群集平均人数
  ρ = ICC (群内相关系数)

ICC 基准值:
  学生-学校 (学业):    0.05–0.25
  患者-诊所 (健康):    0.01–0.05
  工人-企业 (收入):    0.05–0.15
  家庭-邻里:           0.02–0.10
```

### 4.4 功效报告模板

**一手数据**:
> "基于 [来源] 报告的效应量 d=[X]，两-tailed α=.05，80% 功效下，每组最小所需样本量为 N=[X]。目标样本 (N=[Y]) 提供 [Z]% 功效。"

**二手数据**:
> "分析样本 (N=[X]) 在 α=.05 下提供 80% 功效检测 d≥[Y] 的效应。鉴于约束，解释重点放在效应量和置信区间。"

## 五、稳健性方案清单

| 检验 | 目的 | 操作 |
|------|------|------|
| 替代操作化 | X/Y 测量依赖性 | 用替代变量重新估计 |
| 替代样本限制 | 样本构成敏感 | 限制年龄/地域/时间后重估 |
| 加入/移除控制 | 混淆变量敏感 | 逐步加入/移除控制 |
| 排除影响点 | 极端值驱动 | Cook's D > 4/N → 排除重估 |
| 安慰剂结果 | 虚假关联 | 用前处理结果替换 Y |
| 安慰剂处理 | 虚假时机 | DiD: 用假处理时间 |
| Oster (2019) δ | 遗漏变量偏误界 | `sensemakr::sensemakr()` |
| E-value | 未测量混淆界 | `EValue::evalues.OLS()` |

## 六、变量操作化规范

| 角色 | 要求 |
|------|------|
| Y (结果) | 明确测量工具、时间点、分布特征 (均值/SD/范围) |
| X (预测) | 明确操作化方式、效度证据、变异来源 |
| M (中介) | **确认时间顺序**: M 必须在 X 之后、Y 之前测量 |
| W (调节) | 指定交互项形式和分组逻辑 |
| C (控制) | 基于 DAG 选择; **绝不控制**后处理变量或碰撞变量 |

**测量效度清单** (每个关键构念):
- [ ] 表面效度: 操作化是否显然捕捉了概念？
- [ ] 内容效度: 构念的所有面向是否都被覆盖？
- [ ] 建构效度: 该测量是否在此前研究中被验证？引用。
- [ ] 测量等价性: 测量在比较的群体间是否含义一致？
- [ ] 缺失数据: 缺失量多大？最可能的缺失机制 (MCAR/MAR/MNAR)？

## 九、质性方法速查

### 9.1 核心方法
| 方法 | 样本量参考 | 期刊偏好 |
|------|----------|---------|
| 深度访谈 | 20-60人 (以理论饱和为停止标准) | AJS, ASR (混合) |
| 参与观察/民族志 | 数周-数年 | AJS 尤其欢迎 |
| 案例研究 | 1-N 个案例 | ASR, 管理世界 |
| 焦点小组 | 4-8组, 每组6-10人 | 传播学, 公卫 |

### 9.2 抽样策略
| 策略 | 逻辑 | 适用场景 |
|------|------|---------|
| 目的性抽样 | 选择代表关键变异的案例 | 已有初步理论框架 |
| 最大变异抽样 | 覆盖现象的全部变异范围 | 探索性研究 |
| 理论抽样 | 根据新兴理论发展选择 | 扎根理论 |
| 关键案例 | 选择对理论最具检验力的案例 | 检验/挑战既有理论 |

### 9.3 编码策略
**三级编码** (扎根理论):
1. **开放式编码**: 逐行标注 → 100-300个初始编码
2. **轴心式编码**: 合并相关，建立层级 → 15-30个编码家族
3. **选择性编码**: 识别核心范畴，整合所有编码 → 3-5个核心范畴

### 9.4 质性效度策略
- **长期投入**: 足够场域停留时间 → 建立信任和理解
- **三角验证**: 多数据源 × 多方法 × 多研究者
- **参与者确认**: 将初步解读反馈给参与者验证
- **厚描述**: 提供足够背景使读者能评估可迁移性
- **反面案例**: 主动寻找不符合新兴模式的案例
- **审计追踪**: 保留原始数据→分析备忘录→编码决策完整链条

## 十、混合方法设计

### 10.1 五种核心设计
| 设计 | 顺序 | 整合逻辑 |
|------|------|---------|
| **解释性顺序** | QUAN → qual | 量化发现驱动质性追踪 |
| **探索性顺序** | qual → QUAN | 质性发现驱动量化检验 |
| **并行三角验证** | QUAN + qual (同时) | 不同方法验证同一结论 |
| **嵌入式** | QUAN(qual) | 一种方法嵌套在另一种中 |
| **多阶段** | qual→QUAN→qual→QUAN | 多轮迭代 |

### 10.2 整合策略
- **合并 (Merging)**: 两种结果并列 → 寻找趋同和分歧
- **连接 (Connecting)**: 一种方法的结果直接输入另一种
- **嵌入 (Embedding)**: 一种方法为另一种提供背景
- **联合展示 (Joint Display)**: 同表/同图并排呈现两组发现

## 十一、PAP (预分析计划) 模板要点

```
1. 研究问题和假设 (方向性陈述)
2. 设计 (RCT/观测/准实验)
3. 主要结果变量 (名称+操作化+测量+时点)
4. 协变量和控制变量 (列表+每个变量的理由)
5. 主要分析 (估计量+SE类型+软件)
6. 子群/异质性分析 (预指定的调节变量+理由)
7. 稳健性检验 (列出所有计划检查)
8. 多重比较校正 (主假设数K+校正方法)
9. 偏离政策 (偏离PAP将被记录并解释)
10. 样本量 (目标N+功效分析引用)
```
