# 研究设计方法论

> 为 DESIGN 模式提供：将理论框架转化为可操作研究设计蓝图的完整决策体系。

---

## 一、从理论到设计：四步转化链

```
理论锚点确认 → 研究范式选择 → 识别策略匹配 → 操作化与功效 → 理论-方法对齐检验
```

### 步骤 1：理论锚点确认

从 FRAME/STORM 阶段的输出中确认：

| 确认项 | 来源 |
|--------|------|
| 核心理论及其机制链 | frame/ 理论条目中的"机制链条" |
| 理论边界条件 | frame/ 理论条目中的"边界条件" |
| 竞争/替代解释 | frame/ 理论条目中的"竞争/替代理论" |
| 已有实证的效应量基准 | frame/ 理论条目中的"实证证据" |

### 步骤 2：研究范式选择

| 研究目标 | 理论类型 | 推荐范式 |
|---------|---------|---------|
| 检验因果机制 | 中层理论（明确 X→M→Y） | 定量（实验/准实验/观测因果推断） |
| 揭示意义建构过程 | 解释主义/建构论传统 | 质性（深度访谈/民族志/案例研究） |
| 裁定竞争理论 | 存在对立预测的理论对 | 定量（强识别策略：自然实验/IV/RD） |
| 探索新现象/理论建构 | 新兴领域/理论空白 | 质性或混合方法（探索性顺序） |
| 跨层次整合 | 微观-宏观联结理论 | 混合方法或多层次定量 |

### 步骤 3：识别策略匹配

根据数据结构、处理分配方式和理论要求的因果强度，匹配识别策略：

| 理论要求 | 数据结构 | 推荐识别策略 | 关键检验 |
|---------|---------|-------------|---------|
| 无混淆估计 X→Y | 面板/重复测量 | 双向固定效应 | 平行趋势检验（事件研究图） |
| 政策因果效应 | 面板/渐进实施 | 交错 DiD (Callaway-Sant'Anna) | 前趋势检验 + 异质性处理效应诊断 |
| 阈值附近因果 | 截面/面板，规则性断点 | RDD (精确/模糊) | McCrary 密度检验 + 前定变量平衡 |
| 外生变异来源 | 截面/面板，有效 IV | 2SLS / IV | 第一阶段 F > 10 + 过度识别检验 |
| 处理效应 | RCT / 自然实验 | OLS + 稳健 SE | 平衡检验 + 损耗分析 |
| 机制检验 | 面板/实验，M 可测量 | 因果中介分析 (Imai et al.) | 序贯可忽略性假设的敏感性分析 |
| 意义/过程理解 | 质性数据（N=20-60） | 深度访谈/民族志/案例 | 三角验证 + 参与者确认 + 反面案例 |

### 步骤 4：操作化与功效分析

每个关键构念必须明确：
- **理论定义**（来自 frame/ 中的理论条目）
- **操作化指标**（来自目标数据集或自采方案）
- **测量效度证据**（引用已有验证研究）
- **时间顺序**（M 必须在 X 之后、Y 之前）

## 二、七学科常用识别策略偏好

| 学科 | 主流识别策略 | 典型数据来源 | 期刊偏好 |
|------|-------------|-------------|---------|
| **社会学** | FE / DiD / 自然实验 / 混合方法 | GSS, Add Health, PSID, CFPS, CGSS | ASR 偏好强识别+理论深度；AJS 接受质性 |
| **公共管理** | DiD / RD / IV / 案例比较 | 行政数据, 调查数据 | JPART 偏好定量因果；PAR 接受混合方法 |
| **心理学** | 实验（实验室/调查/田野） | 实验数据, 面板调查 | 顶刊偏好多实验+大样本+预注册 |
| **传播学** | 调查实验 / 内容分析 / 面板调查 | 媒体内容数据, 调查面板 | JOC 偏好多方法；CR 接受多种范式 |
| **经济学** | DiD / RD / IV / RCT / 结构估计 | 行政数据, 面板调查 | 顶刊偏好"干净识别"+ 稳健性检验 |
| **教育学** | RCT / 准实验 / 设计研究 | 学校行政数据, PISA, 自采 | AERA 偏好 RCT；定性研究也受认可 |
| **政治学** | 自然实验 / DiD / 调查实验 / IV | 选举数据, 跨国调查, WVS/ESS | APSR 偏好强因果识别；CPS 接受多方法 |

## 三、理论-方法对齐检验（七项）

设计蓝图完成后，通过以下七项检验。任一项"不通过"→ 返回修正：

| # | 检验项 | 通过标准 |
|---|--------|---------|
| 1 | **理论-范式对齐** | 研究范式（定量/质性/混合）与理论的认识论传统一致 |
| 2 | **概念-操作化对齐** | 操作化指标覆盖了理论构念的核心维度（内容效度） |
| 3 | **机制-识别对齐** | 识别策略能区分理论指定的机制与竞争机制的预测 |
| 4 | **边界条件检验** | 研究设计包含对理论边界条件的经验检验或明确讨论 |
| 5 | **竞争解释处理** | 设计中对 frame 文件列出的竞争理论有明确的排除/控制策略 |
| 6 | **功效充分性** | 样本量/信息量满足检测 frame 中报告的典型效应量（≥80% 功效） |
| 7 | **外部效度** | 样本/情境与理论宣称的适用范围一致；若不一致，有明确的局限性讨论 |

## 四、稳健性方案清单（定量设计）

| 检验 | 目的 | 对应理论关切 |
|------|------|-------------|
| 替代操作化 | X/Y 的测量依赖性 | 理论构念是否被特定操作化所绑定 |
| 替代样本限制 | 样本构成的敏感性 | 理论的边界条件/人群适用范围 |
| 加入/移除控制 | 混淆变量的敏感性 | 竞争理论是否通过遗漏变量解释结果 |
| 排除影响点 | 极端值的驱动 | 理论是否仅适用于特定子群 |
| 安慰剂结果 | 虚假关联 | 理论指定的机制是否特定于 Y |
| 安慰剂处理 | 虚假时机（DiD 用） | 处理效应的时间结构是否与理论一致 |
| 敏感性分析 | 未测量混淆的界值 | 竞争解释（frame 中列出的）在多大混淆强度下可推翻结论 |

## 五、DESIGN 输出规范

每次 DESIGN 运行输出：

1. **理论锚点确认**：选定的理论、机制链、边界条件、效应量基准
2. **研究范式与识别策略**：选择理由、关键假设及检验方案
3. **操作化表**：每个构念的理论定义、操作化指标、测量效度证据
4. **功效分析**：基于 frame 中报告的效应量基准计算 MDES
5. **理论-方法对齐检验报告**：七项逐条通过/不通过，不通过项附修正方案
6. **稳健性方案**：列出计划执行的稳健性检验及其对应的理论关切
7. **PAP 概要**：预分析计划的核心要素
